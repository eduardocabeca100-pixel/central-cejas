   - Build: npm install
   - Start: npm start
   - Env vars: Copy from .env (SUPABASE_URL, SUPABASE_KEY, SESSION_SECRET, etc)
   - Note public URL: https://central-cejas-backend.onrender.com

3. Deploy Cloudflare Worker Proxy
   wrangler login
   wrangler secret put BACKEND_URL
   (Paste the Render URL from step 2)
   wrangler deploy

4. Deploy Frontend (Cloudflare Pages)
   - dashboard.cloudflare.com → Pages
   - Connect Git → Select central-cejas
   - Build output directory: /
   - Deploy

❓ TROUBLESHOOTING:

Q: Git push says "CONNECT tunnel failed"?
A: Check internet connection. Try:
   - git push origin main --verbose
   - Or use SSH: git remote set-url origin git@github.com:eduardocabeca100-pixel/central-cejas.git

Q: Backend not responding after deploy?
A: Check Render logs and env variables (SUPABASE_URL, SESSION_SECRET, etc)

Q: CORS errors in frontend?
A: Verify BACKEND_URL in Cloudflare Worker is correct (no trailing slash)

Q: Uploads not working?
A: Backend needs /uploads folder (auto-created). Check Render file system permissions.
